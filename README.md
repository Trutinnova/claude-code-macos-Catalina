  # claude-code-macos-10.15
  
  教程只适用 macOS 10.15（Catalina）。
  
  Claude Code 从 v2.1.113 开始，连 npm 包都换成了原生二进制（需要 macOS 13.0+）。
  
  macOS 10.15.8安装锁定Claude Code v2.1.112（最后一个纯 JS 版本）（npm 装 v2.1.112 最合适）
  
  都是个人安装经验，来分享一下。因为macOS10.15.8系统安装docker4.15版本来安装Claude Code，会有限制，Claude Code无法控制docker以外的事情，
  
  只有在本地安装才可以发挥Claude Code 版本：2.1.112 的全部功能，唯一就是有版本限制，但怎么说都可以使用老系统使用上Claude Code。
